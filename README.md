# 🧮 Calculator Program in C

A simple command-line Calculator Program developed in **C** as part of the **CodeAlpha C Programming Internship**. This calculator performs basic arithmetic operations using the `switch-case` statement.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- Menu-driven interface
- Switch-case implementation
- Division by zero handling
- Easy to understand code

## 🛠️ Technologies Used

- C Programming
- GCC Compiler
- Visual Studio Code

## 📂 Project Structure

```
Calculator_Program/
│── calculator.c
│── README.md
```

## ▶️ How to Compile

```bash
gcc calculator.c -o calculator
```

## ▶️ How to Run

### Windows

```bash
.\calculator.exe
```

### Linux/macOS

```bash
./calculator
```
